Đề bài
LEVEL 1: triển khai deployment chạy nginx (default) lên kubernetes và cho phép truy cập từ bên ngoài thông qua nodePort

output:

+ 1 deployment nginx (replicas=2 pod)
+ 1 nodePort service trỏ tới deployment
+ thực hiên curl tới nodePort và cho ra kết quả trang web mặc định của nginx

Bài làm

1. Tạo file nginx-deployment.yaml
2. Tạo file nginx-service.yaml
3. Tạo deployment: kubectl apply -f nginx-deployment.yaml
4. Kiểm tra deployment tạo ra: kubectl get deployments
-> Kết quả: 
NAME    READY   UP-TO-DATE   AVAILABLE   AGE
nginx   2/2     2            2           16m
5. Tạo service: kubectl apply -f nginx-service.yaml
6 Kiểm tra service tạo ra: kubectl get services
-> Kết quả:
NAME            TYPE        CLUSTER-IP      EXTERNAL-IP   PORT(S)        AGE
nginx-service   NodePort    10.104.82.224   <none>        80:30004/TCP   15m
7. Láy ip của minikube để thực hiện lệnh curl: minikube ip -p star
-> Kết quả: 192.168.49.2
8. Kiể tra bằng lệnh: curl 192.168.49.2:30004
